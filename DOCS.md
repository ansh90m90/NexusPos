# Docs for Nexus

Here are some instructions for generating documentation for Nexus.

## Prerequisites
- [Node.js](https://nodejs.org/) (v16 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

## Docs Commands
1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run docs**:
   ```bash
   npm run docs
   ```

3. **Run docs with watch mode**:
   ```bash
   npm run docs:watch
   ```

4. **Run docs with build mode**:
   ```bash
   npm run docs:build
   ```
