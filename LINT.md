# Lint for Nexus

Here are some instructions for linting Nexus.

## Prerequisites
- [Node.js](https://nodejs.org/) (v16 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

## Lint Commands
1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run lint**:
   ```bash
   npm run lint
   ```

3. **Run lint with fix**:
   ```bash
   npm run lint:fix
   ```

4. **Run lint in watch mode**:
   ```bash
   npm run lint:watch
   ```
