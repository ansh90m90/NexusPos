# Format for Nexus

Here are some instructions for formatting Nexus.

## Prerequisites
- [Node.js](https://nodejs.org/) (v16 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

## Format Commands
1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run format**:
   ```bash
   npm run format
   ```

3. **Run format with fix**:
   ```bash
   npm run format:fix
   ```

4. **Run format in watch mode**:
   ```bash
   npm run format:watch
   ```
