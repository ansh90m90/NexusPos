# Releases for Nexus

All notable releases for this project will be documented in this file.

## [1.0.0] - 2026-04-02
### Added
- Initial release of Nexus.
- Unified POS for retail and restaurant.
- Inventory management with variant support.
- Kitchen Display System (KDS).
- Customer credit system.
- Advanced analytics and reporting.
- Staff management and promotions.
- Responsive design and offline-first architecture.

## [0.1.0] - 2026-03-25
### Added
- Prototype of Nexus with basic POS and inventory features.
- Initial design and layout.
- Basic state management and data handling.
