# Build for Nexus

Here are some instructions for building Nexus.

## Prerequisites
- [Node.js](https://nodejs.org/) (v16 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

## Build Commands
1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

4. **Lint the code**:
   ```bash
   npm run lint
   ```

5. **Test the code**:
   ```bash
   npm run test
   ```
