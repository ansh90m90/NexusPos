# FAQ for Nexus

Here are some frequently asked questions about Nexus.

## General
- **What is Nexus?** Nexus is an all-in-one management application for hybrid retail shops and small-scale restaurants.
- **Is Nexus free?** Yes, Nexus is open-source and free to use.
- **Where is my data stored?** All data is stored locally in your browser's local storage.

## Installation
- **How do I install Nexus?** Check the [README.md](README.md) for installation instructions.
- **What are the prerequisites?** You'll need Node.js (v16 or higher) and npm or yarn.
- **Can I run Nexus without a server?** Yes, Nexus is designed to work as a standalone application.

## Usage
- **How do I use the POS?** Check the [README.md](README.md) for usage instructions.
- **Can I export my data?** Yes, you can export and import your state in **Settings > Data Management**.
- **Is Nexus responsive?** Yes, Nexus is optimized for both desktop and mobile devices.

## Support
- **How do I get support?** Check the [SUPPORT.md](SUPPORT.md) for more information.
- **Can I contribute to Nexus?** Yes, check the [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines.
- **Is Nexus secure?** Yes, check the [SECURITY.md](SECURITY.md) for more information.
