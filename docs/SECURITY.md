# Security Policy

## Supported Versions

We only support the latest version of Nexus. If you are using an older version, please update to the latest version to receive security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < Latest | :x:                |

## Reporting a Vulnerability

If you've discovered a security vulnerability in Nexus, please report it to us as soon as possible. We take security seriously and will investigate all reports.

### How to Report
- Do not open a public issue.
- Send an email to the Nexus Team with a description of the vulnerability and steps to reproduce it.
- Provide as much detail as possible, including screenshots or code snippets if applicable.

### What to Expect
- We will acknowledge your report within 48 hours.
- We will investigate the vulnerability and provide a fix as soon as possible.
- We will keep you updated on the progress of the fix.

Thank you for helping us keep Nexus secure!
